### 데몬 실행
```
sudo launchctl load -w /System/Library/LaunchDaemons/com.apple.atrun.plist
```

### 예약 등록
```
echo "echo hello!" | at 17:16 today
```

### 예약 목록 보기
```
at -l
또는 atq
```

### 삭제
```
at -r 번호
```
